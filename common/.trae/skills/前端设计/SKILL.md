本技能指导创建独特、准生产级的前端界面，避免通用的"AI 粗制滥造"美学。以对美学细节和创意选择的卓越关注，实现真正可运行的代码。

用户提供前端需求：需要构建的组件、页面、应用或界面。可能包含关于用途、受众或技术约束的上下文信息。

## 设计思维

编码前，理解上下文并承诺一个大胆的美学方向：
- **目的**：这个界面解决什么问题？谁使用它？
- **基调**：选择一个极端：极简主义、极繁主义、复古未来主义、有机/自然、奢华/精致、俏皮/玩具感、编辑/杂志风、粗野主义/原始、装饰艺术/几何、柔和/粉彩、工业/实用主义等。有众多风格可选。用它们获取灵感，但设计一个真正符合美学方向的作品。
- **约束**：技术要求（框架、性能、可访问性）。
- **差异化**：什么让这个界面令人难忘？用户会记住的唯一特点是什么？

**关键**：选择清晰的概念方向并精准执行。大胆的极繁主义和精致的极简主义都可行——关键是意图性，而非强度。

然后实现可运行的代码（HTML/CSS/JS、React、Vue 等），确保：
- 准生产级且功能完备
- 视觉冲击力强且令人难忘
- 具有清晰美学观点的凝聚力
- 每个细节都精心打磨

## 前端美学指南

重点关注：
- **字体排版**：选择美观、独特、有趣的字体。避免使用 Arial 和 Inter 等通用字体；选择能提升前端美感的独特字体；出人意料、富有特色的字体选择。将独特的展示字体与精致的正文字体搭配使用。
- **色彩与主题**：坚持统一的美学风格。使用 CSS 变量保持一致性。主色调搭配鲜明的强调色胜过保守、均匀分布的调色板。
- **动效**：使用动画创造效果和微交互。HTML 优先使用纯 CSS 方案。React 项目使用 Motion 库。聚焦高影响力时刻：一个精心编排的页面加载，配合错落的渐显效果（animation-delay）比零散的微交互更能带来愉悦感。使用滚动触发和悬停状态制造惊喜。
- **空间构成**：出人意料的布局。不对称。重叠。对角线流。打破网格的元素。大量的留白或受控的密度。
- **背景与视觉细节**：创造氛围和深度，而非默认使用纯色。添加与整体美学匹配的上下文效果和纹理。应用创意形式，如渐变网格、噪点纹理、几何图案、分层透明、戏剧性阴影、装饰性边框、自定义光标和颗粒叠加。

**绝不**使用通用的 AI 生成美学，如过度使用的字体系列（Inter、Roboto、Arial、系统字体）、陈词滥调的配色方案（尤其是白色背景上的紫色渐变）、可预测的布局和组件模式，以及缺乏上下文特色的模板化设计。

创造性解读并做出出人意料的选择，让设计感觉真正为上下文而生。每个设计都应不同。在明暗主题、字体、美学风格之间变化。**绝不**在不同生成中趋同于常见选择（例如 Space Grotesk 字体）。

**重要**：实现复杂度需与美学愿景匹配。极繁主义设计需要精心编排的代码，包含丰富的动画和效果。极简主义或精致设计需要克制、精准，并仔细关注间距、字体排印和微妙细节。优雅来自于将愿景执行到位。

记住：你有能力创造非凡的创意作品。不要拘束，展现跳出思维框架、全力投入独特愿景时真正能创造出的作品。

## 平台特定设计规范

### 后台管理系统（Web 端）

**设计基调**：现代极简主义 + 微妙的科技感，融合专业感与视觉冲击力。

#### 核心设计特征
- **玻璃态设计**：在关键组件（如头部导航、对话框）使用毛玻璃效果 `backdrop-filter: blur(10px)`
- **动态渐变**：使用精心调配的渐变色，避免单调的纯色块
- **微动效系统**：为交互添加细腻的反馈动画（hover 上浮、聚焦光晕、pulse 动画）
- **数据可视化增强**：图表采用现代化配色和动画效果

#### 布局组件规范

**侧边栏 (Sidebar)**
- 背景：渐变背景 `linear-gradient(180deg, #1e293b 0%, #0f172a 100%)`
- 宽度：展开 260px，收起 72px
- 菜单项：
  - 默认：透明背景，浅灰色文字
  - Hover：10% 透明度白色背景，白色文字
  - 激活：主色渐变背景，白色文字，右侧光晕效果
- 图标：20px，激活时轻微放大动画

**头部导航 (Header)**
- 背景：玻璃态效果 + 半透明白色背景
- 高度：64px
- 边框：底部 1px 半透明边框
- 搜索框：圆角设计，聚焦时主色光晕
- 通知图标：红点 pulse 动画
- 用户头像：圆形，渐变边框

**主内容区 (Main Content)**
- 背景：浅灰色渐变 `linear-gradient(180deg, #f8fafc 0%, #f1f5f9 100%)`
- 内边距：24px
- 卡片间距：24px

#### 业务组件规范

**统计卡片 (StatCard)**
- 背景：白色卡片 + 微妙渐变边框
- 内边距：24px
- 数值：大号字体（32px），使用数字字体，渐变文字效果
- 变化指标：正向绿色背景 + 向上箭头，负向红色背景 + 向下箭头
- 图标：右上角，半透明背景，渐变填充
- Hover：轻微上浮 + 阴影加深
- 动画：页面加载时依次淡入（stagger 动画，间隔 100ms）

**图表卡片 (ChartCard)**
- 背景：白色卡片，圆角 16px
- 标题区：左侧标题，右侧操作按钮
- 图表区：渐变填充区域图，平滑曲线，交互式 tooltip，动画加载
- Hover：卡片上浮 + 阴影加深

**数据表格 (DataTable)**
- 表头：浅灰色渐变背景，加粗深灰色文字，底部边框
- 行：
  - 默认：白色背景
  - Hover：浅蓝色背景 + 轻微上浮
  - 选中：主色背景 + 白色文字
- 单元格：内边距 16px，底部细边框
- 状态标签：不同渐变色背景，圆角设计，小尺寸

#### 表单组件规范

**输入框 (Input)**
- 边框：1px 实线，默认灰色，聚焦时主色
- 圆角：8px
- 内边距：12px 16px
- 聚焦：边框主色 + 光晕（主色 10% 透明度）+ 200ms 过渡

**按钮 (Button)**
- 主按钮：主色渐变背景，白色文字，圆角 8px，Hover 渐变上移 + 阴影，Active 轻微缩放
- 次按钮：透明背景，主色边框和文字，Hover 浅蓝色背景
- 文字按钮：透明背景，主色文字，Hover 下划线

**选择器 (Select)**
- 下拉菜单：白色背景，大阴影，圆角 12px
- 选项 Hover：浅蓝色背景
- 选项选中：主色背景 + 白色文字

#### 反馈组件规范

**对话框 (Dialog)**
- 背景：白色 + 毛玻璃遮罩
- 圆角：20px
- 阴影：大阴影
- 动画：scaleIn + fadeIn 组合

**消息提示 (Message)**
- 成功：绿色渐变背景 + 对勾图标
- 警告：橙色渐变背景 + 警告图标
- 错误：红色渐变背景 + 错误图标
- 信息：蓝色渐变背景 + 信息图标
- 动画：从顶部滑入 + 淡出

**加载状态**
- 全屏加载：半透明白色 + 模糊，旋转渐变圆环
- 骨架屏：浅灰色渐变背景，shimmer 效果

#### 页面模式规范

**登录页面**
- 背景：全屏渐变 + 动态粒子效果
- 登录框：居中，白色卡片 + 大阴影，圆角 24px，宽 420px
- 标题：大号字体，渐变文字
- 表单：大尺寸输入框，聚焦光晕，全宽按钮
- 动画：登录框从下方淡入，输入框标签聚焦上浮

**仪表盘页面 (Dashboard)**
- 欢迎区域：主色渐变背景，内边距 40px，顶部圆角 20px，白色文字，三列统计数据
- 统计卡片区：4 列网格（响应式），stagger 动画依次淡入
- 图表区：2 列网格，白色卡片大圆角
- 最近订单：现代化表格，按钮组操作

**列表页面（用户管理、角色管理等）**
- 筛选区：白色卡片，内边距 20px，圆角 16px，多列表单 + 搜索 + 操作按钮
- 表格区：白色卡片，内边距 0，圆角 16px
- 分页区：表格底部，简约设计，圆形页码按钮

**表单页面（新增/编辑）**
- 布局：复杂表单单列，简单表单双列
- 卡片分组：每个分组白色卡片，标题左侧加粗，内边距 24px
- 表单项：标签右对齐（桌面端），大尺寸输入框，小号浅灰提示文字
- 底部操作栏：页面底部固定，白色背景 + 阴影，按钮右对齐

**详情页面**
- 头部信息卡片：白色卡片，左侧基本信息，右侧操作按钮，内边距 24px
- 标签页内容区：白色卡片，卡片式标签页，内边距 24px
- 信息展示：键值对列表，网格布局，浅灰标签，深灰加粗值

**错误页面（404, 500）**
- 背景：渐变背景
- 内容居中：超大号错误代码（120px）渐变文字，中号灰色描述，主按钮返回首页
- 插图：SVG 插画，浮动动画

#### 响应式规则

**断点定义**

**移动端适配（< 768px）**
- 侧边栏：默认收起，点击菜单按钮弹出，黑色半透明遮罩
- 表格：横向滚动或卡片式布局
- 表单：单列布局，全宽按钮
- 统计卡片：1-2 列布局

**平板适配（768px - 1024px）**
- 侧边栏：正常展开
- 表单：双列布局
- 统计卡片：2-3 列布局

**桌面适配（> 1024px）**
- 统计卡片：4 列布局
- 图表：并排显示
- 表格：完整显示

#### 暗黑模式适配

**色彩调整**
- 背景：深蓝系（#0f172a, #1e293b, #334155）
- 文字：浅灰系（#f1f5f9, #cbd5e1, #64748b）
- 边框：深灰系（#334155, #475569）
- 主色：降低饱和度，提高亮度
- 阴影：更高透明度黑色阴影

- 玻璃态效果：增强模糊效果
- 渐变：使用更深的渐变色
- 光晕：降低强度
- 对比度：确保文字可读性（至少 4.5:1）

## 性能与可访问性

### 性能优化

**CSS 优化**
- 使用 CSS 变量实现主题切换
- 避免过度使用 box-shadow 和 filter
- 使用 will-change 优化动画性能
- 使用 transform 代替 top/left 进行动画
- 避免过度嵌套选择器

**资源优化**
- 图标使用 SVG Sprite
- 图片使用 WebP 格式
- 字体使用子集化
- 按需加载组件样式
- 图片懒加载

**动画优化**
- 使用 CSS 动画代替 JavaScript 动画
- 使用 requestAnimationFrame
- 避免同时触发大量动画
- 提供减少动画选项
- 确保动画帧率 > 50fps

**性能指标**
- 页面加载时间 < 2 秒
- 首次内容绘制 < 1.5 秒
- 可交互时间 < 3 秒
- Lighthouse 性能分数 > 90

### 可访问性

**键盘导航**
- 所有交互元素可聚焦
- 清晰的焦点样式
