# 主题系统使用指南

## 📋 概述

IxPay Pro Web 项目已经建立了完整的**白天/黑暗模式切换系统**,采用 Element Plus + Tailwind CSS + 自定义设计系统的混合架构，提供强大的主题定制能力。

## 🎨 主题系统架构

### 核心特性

- ✅ **三种主题模式**: 浅色 (light) / 深色 (dark) / 自动 (auto)
- ✅ **动态主题色**: 支持 8 种预设主题色
- ✅ **CSS 变量系统**: 完整的配色、间距、圆角、阴影系统
- ✅ **持久化存储**: 用户偏好自动保存
- ✅ **响应式设计**: 完美适配各种屏幕尺寸
- ✅ **无障碍支持**: 高对比度、减少动画模式

### 技术栈

- **状态管理**: Pinia + @vueuse/core
- **CSS 框架**: Tailwind CSS 3.4.17
- **UI 组件库**: Element Plus 2.11.2
- **CSS 预处理器**: Sass 1.92.1

## 🚀 快速开始

### 1. 主题切换按钮

在任意组件中添加主题切换功能:

```vue
<template>
  <el-switch
    v-model="appStore.isDark"
    @change="handleThemeChange"
    :active-action-icon="Moon"
    :inactive-action-icon="Sunny"
  />
</template>

<script setup lang="ts">
import { useAppStore } from '@/stores/modules/app'
import { Moon, Sunny } from '@element-plus/icons-vue'

const appStore = useAppStore()

const handleThemeChange = (value: boolean) => {
  appStore.isDark = value
  // 主题会自动切换
}
</script>
```

### 2. 使用 Tailwind CSS 暗黑模式

```vue
<template>
  <div class="bg-white dark:bg-gray-800">
    <h1 class="text-gray-900 dark:text-white">标题</h1>
    <p class="text-gray-600 dark:text-gray-300">内容</p>
  </div>
</template>
```

### 3. 使用 CSS 变量

```vue
<template>
  <div class="custom-card">
    <h3>卡片标题</h3>
  </div>
</template>

<style scoped lang="scss">
.custom-card {
  background-color: var(--bg-light);
  color: var(--text-primary);
  border: 1px solid var(--border-color);
  box-shadow: var(--shadow-md);

  &:hover {
    box-shadow: var(--shadow-lg);
  }
}
</style>
```

## ♿ 无障碍支持

### 高对比度模式

```scss
@media (prefers-contrast: high) {
  :root {
    --text-primary: #000000;
    --border-color: #000000;
  }

  html.dark {
    --text-primary: #ffffff;
    --border-color: #ffffff;
  }
}
```

### 减少动画模式

```scss
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

## 📊 主题状态管理

### Pinia Store

主题状态通过 Pinia store 管理:

```typescript
// stores/modules/app.ts
interface ConfigType {
  weakness: boolean // 色弱模式
  grey: boolean // 灰度模式
  primaryColor: string // 主题色
  showTabs: boolean // 标签栏显示
  darkMode: string // 主题模式 (light/dark/auto)
  layout_side_width: number // 侧边栏宽度
  side_mode: string // 侧边栏模式
  transition_type: string // 过渡动画类型
}
```

### 使用方法

```typescript
import { useAppStore } from '@/stores/modules/app'

const appStore = useAppStore()
const { config, isDark, currentTheme } = storeToRefs(appStore)

// 切换主题
appStore.toggleLightMode()
appStore.toggleDarkModeForce()
appStore.toggleAutoTheme()

// 切换主题色
appStore.togglePrimaryColor('#722ED1')

// 切换特殊模式
appStore.toggleWeakness(true)
appStore.toggleGrey(false)

// 重置配置
appStore.resetConfig()
```
